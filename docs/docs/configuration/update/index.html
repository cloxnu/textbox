    Update #

    If you want to change the content or style of the textbox, try update.

    Overview # 

    textbox({
    title: 'Original Title',
    btn_text: 'Change Title',
    btn_callback(tb) {
        tb.update({
            title: 'Updated Title',
        });
    },
});
